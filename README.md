# SE-Assignment
Order Management App

**Frontend-side tools and technologies used**

    •	React

    •	Modern Javascript(ES6)

    •	NodeJS and NPM

    •	VS Code IDE

    •	Create React App CLI

    •	Axios HTTP Library



**Server-side Technologies used**

    •	Spring Boot

    •	SpringData JPA

    •	Maven 3.2+

    •	JDK 1.8

    •	Embeded Tomcat 8.5+

    •	MySQL Database
